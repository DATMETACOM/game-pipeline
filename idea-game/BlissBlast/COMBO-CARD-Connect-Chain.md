## 📋 COMBO CARD FORMAT

## COMBO-001: CONNECT CHAIN

- **Base product:** Flow Free (Connect core)
- **Donor 1:** Mr Bullet (Chain reaction payoff)
- **Donor 2:** Parking Jam (Cascading clears)
- **Core:** Connect/Draw with chain reaction trigger
- **Twist:** Connection causes cascade, not just completion
- **Payoff:** Chain reaction + release + flow
- **Theme:** Transport (pipes/wires) or Household (plumbing)
- **3-second hook:** Draw line → Watch chain reaction → Level clears
- **Because proven:** Connect core (Flow Free) + Chain reaction (#3 CPI hook)
- **Clone risk:** Medium (easy to understand, harder to polish)

## 🔍 FRAME REFERENCE RESEARCH

### CATEGORY A: BASE PRODUCTS (Connect Core)

#### REF-A1: Flow Free
**Downloads:** 100M+ | **Rating:** 4.5★

| Aspect | Details |
|--------|---------|
| **Core** | Connect same-colored dots with pipes without crossing |
| **Action** | Draw/swipe to create path |
| **Constraint** | No crossing paths, fill entire grid |
| **Payoff** | Flow sensation when path connects |
| **Difficulty** | Grid size increases, more color pairs |
| **IAP Fit** | Medium (hints, level skips) |

**Why Proven:**
- Clear 3-second readability
- Satisfying "flow" payoff
- Progressive difficulty
- Multiple solution paths

**Visual Reference:**
```
Grid 5x5 with colored dots:
🔵----🔵
🔴----🔴
🟢----🟢
```

### CATEGORY B: DONOR PRODUCTS (Chain Reaction Experience)

#### REF-B1: Mr Bullet (RECON-003)
**Downloads:** 100M+ | **Rating:** 4.6★

| Aspect | Details |
|--------|---------|
| **Core** | Aim and shoot, bullet follows physics |
| **Twist** | Physics puzzle + tactical genius fantasy |
| **Action** | Tap to aim |
| **Payoff** | Ricochet, ragdoll, **chain kills** |
| **Chain Reaction** | One bullet hits multiple enemies |
| **IAP Fit** | Medium (power-ups, hints) |

**Key Learning:**
> **Chain reaction = strong CPI hook**
> One input (bullet) → Multiple outputs (kills) → User feels smart

**Chain Reaction Mechanics:**
```
Bullet → Enemy 1 → Ricochet → Enemy 2 → Explosion → Enemy 3
        (Kill #1)           (Kill #2)             (Kill #3)
```

**Visual Feedback:**
- Slow-mo on critical shots
- Number counter: 1... 2... 3...
- Screen shake on impact
- Combo multiplier

## 🎯 TARGET COMBINATION FOR IDEA 1

### RECOMMENDED: COMBO-C1 (Connect + Explosion)

**Why This Combo:**
1. ✅ Proven core (Flow Free connect)
2. ✅ Proven payoff (Mr Bullet chain reaction)
3. ✅ Clear 3-second hook
4. ✅ High IAP fit (explosions = monetization moments)
5. ✅ Easy to prototype
6. ✅ Mass market appeal (transport theme)

**Core Loop:**
```
1. User draws pipe connection
2. Liquid/power starts flowing
3. Reaches bomb/booster
4. BOOM! Cascade clear
5. Obstacles destroyed
6. Path to exit opens
7. Level complete celebration
```

**Fail States (Near-Win Design):**
- 95% progress → Bomb doesn't explode (wrong connection)
- 90% progress → Liquid stops 1 pipe from exit
- 89% progress → Explosion clears everything except one block

**APS Target:** 1.4-1.6
- Multiple ways to connect
- Strategic bomb placement
- Planning required but fair
