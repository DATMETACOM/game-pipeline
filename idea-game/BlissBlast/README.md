## 💡 Key Insights from Research

1. **Chain reaction is magic**
   - One input → Multiple outputs
   - User feels smart/tactical
   - Highly shareable in ads

2. **Cascading clears = dopamine**
   - Parking Jam proved this works
   - One move → Multiple rewards
   - "Everything falling into place" feeling

3. **Near-win design is critical**
   - 89-95% progress moments
   - "Just one more pipe" frustration
   - Creates IAP motivation

4. **Combo multiplier = engagement**
   - Toon Blast proved this
   - x2, x3, x4 increases satisfaction
   - Visual + audio feedback essential
